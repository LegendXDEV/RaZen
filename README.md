## Self-hosting

click the download button and in `.env` place your discord bots token. example:

```json
{
  "token": "YOUR_BOT_TOKEN"
}
```
make sure you have [node.js](https://nodejs.org/en/)

to run the bot use `node index` or `node index.js`
